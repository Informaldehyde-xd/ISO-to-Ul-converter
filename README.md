This App is made for Android to convert iso to ul file offline
It also Makes the ul.cfg file for opl to read.
SETUP---
1. Simply download the apk and choose the iso you want to convert
2. make sure the iso must be in the Download or Document folder on your mobile
3. Also make the PS2_GAME folder in the same directory (Download or Document)W
4. on converting export the files to PS2_GAME folder
5. the ul and ul.cfg will be there after conversion.
6. You can also convert the ul back to iso from the same method from adjacent tab

IMPORTANT NOTE
1. THE APP IS IN DEVELOPMENT STATE AND TESTING SO IT CAN BE LITTLE FLICKY
2. WHILE CONVERTING IF IT SHOWS SOME ERROR JUST CHANGE THE EXPORT PATH BETWEEN DOWNLOAD, SUB FOLDERS IN DOWNLOAD OR THE DOCUMENT FOLDER
